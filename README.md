> The no-brainer ImageMagick CLI for optimizing images

## Install

```bash
$ npm install optimo --global
```

## CLI

```bash
$ optimo --help

  Usage
    $ optimo <file> [options]

  Options
    -d, --dry-run  Show what would be optimized without making changes

  Examples
    $ optimo image.jpg
    $ optimo image.png --dry-run
    $ optimo image.jpg -d
```
